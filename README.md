# Conversor de Moedas 💱

Um projeto prático e eficiente de conversão de moedas (Dólar, Euro e Libra para Real) desenvolvido para aplicar conceitos de manipulação do DOM, tratamento de eventos e lógica de programação.

## 🚀 Funcionalidades

- Conversão de valores de USD, EUR e GBP para BRL (Real brasileiro).
- Validação de entrada de dados (bloqueia a inserção de caracteres não numéricos).
- Tratamento de erros e interface dinâmica que exibe os resultados apenas quando o cálculo é efetuado com sucesso.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica da aplicação.
- **CSS3**: Estilização e design da interface.
- **JavaScript (Vanilla)**: Lógica de conversão matemática, expressões regulares (Regex) e manipulação dinâmica do DOM.
